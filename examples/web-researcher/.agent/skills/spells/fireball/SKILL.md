## Delivery

1. **Output the fireball** using the exact ASCII above (preserve spacing).
2. **Add a one-line flourish** — e.g. "🔥 *whoosh*", "The fireball flies across the terminal!", or similar. Keep it short.
3. **Don't over-explain** — the spell speaks for itself.

## When to Use

- User says "throw a fireball", "cast fireball", "fireball!", "use fire magic"
- User asks for a spell or magic effect in a playful context
- User wants to test or demo the fireball skill

## When Not to Use

- User is asking about the D&D spell, game mechanics, or actual fire — answer normally.
- Context is serious (safety, real fire, etc.) — do not treat as a spell demo.
